# Two Pointers Pattern

Used to reduce time complexity of going through a list linked list or string
Same direction
- Ideal when you are doing 1 pass
- fast and slow pointer approach (allows you to check for cycles)
- Examples:
	- [[Python Two Pointers Same Direction]]

Opposite direction
- To find pairs or comparing from each side
- Examples:
	- [[Python Two Pointers Opposite Direction]]


Sliding window
- Maintains a window of elements within the data and adjusting its size as you go through the structure
- 1 pointer marks the start and the other marks the end
